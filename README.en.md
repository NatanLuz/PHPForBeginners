# PHP for Recruits

[![Portuguese](https://img.shields.io/badge/lang-Portuguese-009739)](README.md) [![English](https://img.shields.io/badge/lang-English-1E90FF)](README.en.md)

A PHP study repository focused on hands-on learning, original examples, and progressive practice.

This project is designed for beginners who want to learn PHP in a clear, organized, and practical way. 🚀

Portuguese version: [README.md](README.md)

## 1. Learning Methodology

This repository follows an active learning approach:

- study each concept and practice it immediately;
- create original examples instead of copying ready-made code;
- review and refactor examples to reinforce fundamentals;
- solve topic-based exercises to validate understanding.

Methodology goals:

- build a solid foundation in PHP fundamentals;
- understand why each solution works;
- prepare for real-world project scenarios.

## 2. Folder Structure

Main structure in English, standardized with snake_case:

- [arrays](arrays): array creation and manipulation.
- [constants](constants): constants and fixed configuration values.
- [control_structures](control_structures): if, else, switch, and match.
- [loops](loops): repetition structures such as for and while.
- [functions](functions): function definitions, parameters, and returns.
- [strings](strings): string handling and transformations.
- [variables](variables): variables, types, and assignments.

Exercise structure by topic:

- [exercises/arrays](exercises/arrays)
- [exercises/functions](exercises/functions)
- [exercises/loops](exercises/loops)

Pattern used in each topic folder:

- numeric-prefix examples: 01*, 02*, 03\_...
- local README.md explaining the topic goal
- exercise1.php and exercise2.php for practice

## 3. Covered Topics
